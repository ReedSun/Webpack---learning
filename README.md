# Webpack---learning
I am learning webpack. Fighting!
## demo1
我做的工作：
- 设置npm镜像源
```
npm config set registry https://registry.npm.taobao.org/
```
- 安装webpack
```
npm i -g webpack
```
*安装完成后可以通过`webpack --help`验证是否安装成功*
- 安装一些npm包
```
npm i babel-loader
npm i babel-core
babel-preset-es2015
npm i babel-preset-react
```
- 运行成功~
